SIGNO ("we", "our", or "us") values your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use the SIGNO mobile application.

1. Information We Collect

Account Information

When you create an account, we may collect:

  • Email address
  • Authentication information (including Sign in with Apple, if selected)
  • Encrypted or hashed passwords

User Content

We store content created within the app, including:

  • Custom sign images uploaded by you
  • Assigned meanings or labels
  • Learning progress
  • Module completion status
  • Practice history

Usage Data

We may store limited usage data required to operate core app features.

2. How We Use Information

We use the collected information to:

  • Create and manage user accounts
  • Save learning progress
  • Store custom signs and assigned meanings
  • Provide translation and learning features
  • Improve overall app functionality

We do not sell your personal information.

3. Third-Party Services

SIGNO uses Supabase as a backend infrastructure provider to securely store and manage user data. Supabase processes data solely to provide backend services.

4. Data Security

Passwords are stored in encrypted or hashed form and are never stored in plain text. We implement reasonable security measures to protect user data.

5. Data Retention and Deletion

User data is retained while the account remains active. Users may request account deletion by contacting us. Upon deletion, associated data will be permanently removed.
